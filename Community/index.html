Community

Wiki

The user editable Sysmo Wiki. A must before going further…

Administrators questions

Any new question matching the tags sysmo or sysmo-nms on the serverfault.com service will alert us. Post any question concerning installation and configuration here. We will reply as soon as we can.

Developpers questions

Any new question matching the tags sysmo, sysmo-nms or nchecks on the stackoverflow.com service will alerts us. Post any question concerning the extensions API here. (Take a look at the Wiki before !).

sysmo-nms-users mailing list

This mailing list should be used for Feature request or general discussion about Sysmo. Because of the better visibility of Serverfaults or Stackoverflow in most search engines, these are the preferred way to ask questions you think can be useful for others.

sysmo-nms-announce mailing list

If you only want to be notified of new releases, subscribe to this list.

Bug Tracker

Users are encouraged to report any bug, however insignificant it may seem. Please run a couple of searches first to see if your bug has been reported.
